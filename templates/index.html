Bevy is an open-source modular game engine built in Rust, with a focus on developer productivity and performance.
Data Driven and Massively Parallel
All engine and app logic builds on top of Bevy's blisteringly fast Entity Component System.
Modern Renderer
  • Multiple Backends: Vulkan, DirectX 11/12, Metal
  • Render Graph: Define your render logic in an intuitive, hyper-parallel, graph system
  • Data Driven Shaders: Powerful data driven shader system that integrates directly with Bevy's Entity Component System
Expressive UI
Easily compose UIs using a familiar "box model". Bevy's UI system is custom-built to integrate directly into Bevy's ECS and event systems.
Cross Platform
Write once, run anywhere!
  • Supported Platforms: Windows, MacOS, Linux
  • Upcoming Platforms: Android, Web
Fast Iterative Compile Times
Bevy has fast iterative compiles: projects can often be compiled in 1-2 seconds. You don't need to choose between slow but productive scripting languages or fast but unproductive compiled languages. Write fast, native code productively directly in Rust!
Free and Open Source
Bevy is developed in the open by a community of volunteers. You own the code. You don't ever need to pay us anything. But feel free to contribute code if you built something cool!
